<ul><li><strong>ScriptHookDotNet & ASI-Management:</strong> Vollständige Kontrolle über .NET-Skripte und ASI-Plugins – die Grundlage für die meisten GTA IV-Mods. Installieren und aktivieren Sie sie mit einem Klick.</li>

<h3>⚙️ So verwenden Sie GTAModsManager mit GTA IV:</h3>
<ul><li><strong>Download & Installation:</strong> Entpacken Sie die ZIP-Datei und führen Sie GTAModsManager.exe aus.</li>
<li><strong>GTA IV-Ordner auswählen:</strong> Wählen Sie Ihr GTA IV-Verzeichnis (Steam, Rockstar Launcher oder Disc-Version).</li>
<li><strong>Baseline erstellen:</strong> Lassen Sie den Manager beim ersten Start eine Referenz der Originaldateien erstellen.</li>
<li><strong>Profil erstellen:</strong> Erstellen Sie ein neues Profil (z.B. "Main Profile" oder "EFLC").</li>
<li><strong>Mods hinzufügen:</strong> Ziehen Sie Mod-Archive (ZIP, RAR, 7z) per Drag-and-Drop in das Manager-Fenster oder legen Sie sie manuell im entsprechenden Profil-Ordner ab.</li>
<li><strong>Mods installieren & aktivieren:</strong> Installieren Sie die Mods über den Manager und aktivieren Sie sie per Häkchen.</li>
<li><strong>Spiel starten:</strong> Starten Sie GTA IV wie gewohnt – der Manager sorgt dafür, dass nur aktive Mods geladen werden.</li></ul>
